Project 2 - Fun with Filters and Frequencies

Filip Malm-Bägén

This project explores various techniques for processing and enhancing images through the use of filters and frequencies. First, it demonstrates edge detection using finite difference operators and smoothing filters like the derivative of Gaussian (DoG). Then, the project delves into image sharpening through unsharp masking by emphasizing high-frequency components. Hybrid images are created by blending the high frequencies of one image with the low frequencies of another, producing visually dynamic results. Finally, the project explores multiresolution blending using Gaussian and Laplacian stacks to seamlessly merge images across different frequency bands.

This webpage design was partly made with generative AI models.
